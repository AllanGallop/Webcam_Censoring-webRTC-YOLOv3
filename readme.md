## Requirements
> OpenCV
> Cuda
> CuDNN
> Darknet
> Yolo

## Install AIORTC
``` apt install libavdevice-dev libavfilter-dev libopus-dev libvpx-dev pkg-config libsrtp2-dev ```

``` pip3 install aiortc ```

## Run
Start Server
``` python3 server.py ```

Open webpage & allow webcam
``` localhost:8080 ```
